# AI-Интегратор 2026 ⚡

**Интерактивная образовательная платформа для обучения AI-автоматизациям на базе Make.com.**

Программа «Путь к первому доходу за 90 дней» разработана для обучения специалистов по внедрению нейросетей в бизнес-процессы. От базового понимания алгоритмов до создания сложных корпоративных AI-агентов с памятью.

![Platform Preview](https://img.youtube.com/vi/placeholder/0.jpg)

## 🚀 Быстрый старт

Платформа реализована как **SPA (Single Page Application)** на чистом стеке (HTML/CSS/JS) и не требует настройки базы данных или сервера.

1.  **Локальный запуск**: Используйте `npx serve .` или расширение Live Server в VS Code.
2.  **Простой просмотр**: Откройте `index.html` — всё работает "из коробки" благодаря глобальному подключению скриптов.

## 📚 Программа обучения

1.  **🧠 Мышление Интегратора** — Логика нейросетей и основы API.
2.  **🔗 Первые связи в Make** — Автоматизация Google Sheets + Telegram.
3.  **🤖 Мозги для робота** — Промпт-инжиниринг и OpenAI API.
4.  **💾 AI-Агенты и Память** — Построение RAG-систем на Airtable.
5.  **💰 Бизнес и Продажи** — Упаковка кейсов и продажи AI-решений.

## 💎 Особенности v2.0
-   **Премиальный UI**: Анимированные фоновые орбы и стеклянный дизайн.
-   **Геймификация**: Наглядная XP-система с анимированными счетчиками на главном дашборде.
-   **Умные Тесты**: Поддержка клавиатуры (1-5, Enter), анимированные кольца результатов и разбор ошибок.
-   **Плавная Навигация**: Направленные переходы между страницами (налево/направо).
-   **Производительность**: Мемоизация состояния и Shimmer-эффекты при загрузке.

## 🛠 Технический стек
-   **Core**: Vanilla JS (Modern ES6+).
-   **Styling**: CSS3 (Variables, Grid, Flexbox, Keyframe Animations).
-   **State**: LocalStorage Wrapper with cache.
-   **Graphics**: Canvas API (Confetti & Progress Rings).
