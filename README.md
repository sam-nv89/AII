# AI-Интегратор 2026 ⚡

**Интерактивная образовательная платформа для обучения AI-автоматизациям на базе Make.com.**

Программа «Путь к первому доходу за 90 дней» разработана для обучения специалистов по внедрению нейросетей в бизнес-процессы. От базового понимания алгоритмов до создания сложных корпоративных AI-агентов с памятью.

![Platform Preview](https://img.youtube.com/vi/placeholder/0.jpg)

## 🚀 Как запустить

Платформа реализована как **SPA (Single Page Application)** на чистом стеке (HTML/CSS/JS) и не требует установки зависимостей или настройки базы данных.

### Вариант 1: Простое открытие
Просто откройте файл `index.html` в вашем любимом браузере. Благодаря отсутствию модульных импортов, всё будет работать прямо из файловой системы.

### Вариант 2: Локальный сервер (Рекомендуется)
Для лучшей производительности и корректной работы всех API-функций в будущем:
- Используйте VS Code расширение **Live Server**.
- Или запустите через Node.js: `npx serve .`
- Или через Python: `python -m http.server`

## 📚 Структура курса

Программа разделена на 5 ключевых этапов (Модулей):
1. **🧠 Мышление Интегратора** — Основы AI, API и логика HTTP-запросов.
2. **🔗 Первые связи в Make** — Работа с Google Sheets, Telegram Bot API и Webhooks.
3. **🤖 Мозги для робота** — Продвинутая интеграция OpenAI, промпт-инжиниринг и логические фильтры.
4. **💾 AI-Агенты и Память** — Построение RAG-систем (память на базе Airtable) и Vision API.
5. **💰 Бизнес и Продажи** — Поиск клиентов, аудит процессов и ценообразование услуг.

## 💎 Особенности платформы
- **XP-система**: Геймифицированное обучение с уровнями и очками опыта.
- **Персистентность**: Ваш прогресс сохраняется в браузере (localStorage) — вы не потеряете данные при перезагрузке.
- **Интерактивные тесты**: 50 контрольных вопросов с мгновенной проверкой знаний.
- **Премиальный UX**: Современный темный интерфейс с плавными анимациями и адаптивной версткой.

## 🛠 Технологии
- **Frontend**: Pure HTML5, CSS3 (Modern Grid/Flexbox), Vanilla JS.
- **State**: Браузерное хранилище (localStorage).
- **Animations**: CSS Keyframes + Canvas Confetti Engine.
