# Архитектура Платформы AI-Интегратор 2026

Документ описывает техническое устройство и логику работы SPA-приложения.

## 🏗 Общая структура (SPA)

Приложение построено на базе **Single Page Application** модели без использования тяжелых фреймворков. Это обеспечивает мгновенную загрузку и легкость поддержки.

### Основные компоненты системы:
1.  **Router (`js/app.js`)**: Централизованный контроллер. Обрабатывает изменения хэша (`#home`, `#module-N`, `#quiz-N`). Включает **Route Guard** для проверки доступности контента и управляет **Directional Transitions** (анимация сдвига влево/вправо в зависимости от направления навигации).
2.  **State Manager (`js/store.js`)**: Управляет данными пользователя. Использует **Memoization** (кэш `_cache`) для уменьшения нагрузки при частых обращениях к `localStorage`. Инвалидация кэша происходит автоматически при каждой записи.
3.  **Modules Data (`js/modules/`)**: Контент курса. Содержит теорию, шаги воркшопа и вопросы тестов.
4.  **View Renderers (`js/views/`)**: Декларативные функции рендеринга HTML (Home, Модули, Тесты).

## 💾 Схема данных (Store)

Данные хранятся в объекте (синхронизация с `localStorage`):
```javascript
{
  xp: 1250,                 // Общий опыт
  completedModules: [1, 2], // Пройденные модули
  quizScores: { "1": 90 },  // Результаты тестов
  currentModule: 2,         // Последний активный модуль
  startDate: 1713180000000  // Дата начала (Unix timestamp)
}
```

## 🎨 Дизайн-система (CSS v2.0)

Иерархия стилей для максимальной гибкости:
-   `css/reset.css`: Нормализация.
-   `css/tokens.css`: Глобальные переменные (палитра, сетка отступов, радиусы, тени, z-индексы).
-   `css/layout.css`: Сетка приложения. Включает **Ambient background orbs** (анимированные фоновые элементы).
-   `css/components.css`: Стили атомарных компонентов. Включает расширенный набор состояний (hover glow, shimmer loading).

## 🧩 Система квизов (Quiz Engine v2.0)

Реализована в `js/views/quiz.js`:
1.  **Scoped State**: Таймеры и промежуточные ответы изолированы от глобальной области.
2.  **Keyboard Navigation**: Поддержка клавиш `1-5` для выбора и `Enter/Backspace` для навигации.
3.  **Animated Results**: Canvas-анимация кольца успеха и динамический счетчик XP.
4.  **Review System**: Анимированный слайдер разбора ошибок с динамической подстройкой высоты.

## 📈 XP и Уровни

Система званий (рассчитывается в `getLevelInfo()`):
